# DevConnect

DevConnect is a Vite + React experience that pairs developers and users through a sleek, animated landing page, role-based onboarding, and a mock developer dashboard. It ships with Tailwind CSS styling, React Router v6 navigation, and a dark/light mode toggle.

## Features

- Animated landing page with search and CTA
- Role-based auth flow (login/signup) with inline validation
- Forgot password screen with success state
- Developer dashboard with mock data cards
- Responsive navbar with mobile menu
- Dark/light mode toggle stored in localStorage
- Smooth route transition fade

## Setup

```bash
npm install
npm run dev
```
